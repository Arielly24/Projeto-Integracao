FPOO

Linguagem

Em fundamentos de programação orientado a objeto, usamos a linguagem de programação "JAVA" onde vimos classes, métodos, if e else, get e set, concatenação e etc. Aprendemos a fazer diversos projetos como calculadora de IMC, estacionamento e até mesmo um jogo de jokempô.

Aplicação

Para o estudo dessa linguagem utilizamos o aplicativo Eclipse, onde temos diversas ferramentas do Java, facilitando a programação pois os erros são mostrados de imediato ao invés de serem mostrados apenas no compilador.

LIMA

Projetos

Em linguagem de marcação vimos diversas maneiras de criar layouts, editar cores, formatos, imagens, colunas e etc. Fizemos dois projetos muito interessantes que foram o Comida e Cidade invisível onde reproduzimos a tela de uma série da Netflix.

Linguagem

Para essa linguagem utilizamos o HTML e o CSS, onde vimos div, section, display flex, display grid, backgroung-color, padding, margin e muitos outros comandos para a edição do nosso layout. Fizemos um gif dentro do layout do cidade invisível dando impressão de movimento na imagem.

SOP

Aprendizado

Se tratando de sistemas operacionais, aprendemos sobre os mesmos, suas origens e criações, versões, evoluções, interface gráfica, tabela de valores e etc. Vimos também sobre a instalação de um sistema operacional um uma máquina virtual, usando a ferramenta VirtualBox. E, por fim, também aprendemos a utilizar algumas ferramentas do Excel.
